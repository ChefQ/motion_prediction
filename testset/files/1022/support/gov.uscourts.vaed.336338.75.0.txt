DEFENDANT'S MOTION TO SET ASIDE DEFAULT

JUDGMENT RE FXF.COM

Ye Yu, erroneously sued as John Doe (re domain FXF.com, section 10 of the
complaint), pursuant to Federal Rule of Civil Procedure 60(b), hereby request to

set aside the Default Judgment entered on May 18,2016 (ECF 43).
Pursuant to Local Civil Rule 7(E), Ye Yu contacted Plaintiff's counsel in

good faith to discuss the ground to this motion and requested that the default
judgment be set aside.

The basis for this motion is set forth more fully in the accompanying
Memorandum in Support of Motion to Set Aside the Default Judgment.
